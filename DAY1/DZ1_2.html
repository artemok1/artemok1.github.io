Additional Tags

<dl>, <dt>, <dd> tags

The <dl> tag defines a description list.
The <dl> tag is used in conjunction with <dt> (defines terms/names) and <dd> (describes each term/name)

E.g.:
<dl>
  <dt>Term1</dt>
  <dd>Description of term 1</dd>
  <dt>Term2</dt>
  <dd>Description of term 2</dd>
</dl>

will result in:

Term1
Description of term 1
Term2
Description of term 2

<pre> tag

The <pre> tag defines preformatted text.
Text in a <pre> element is displayed in a fixed-width font (usually Courier), and it preserves both spaces and line breaks.

E.g.:
<pre> On this site you will find versions of some classics of early modern philosophy, and a few from the 19th
century, prepared with a view to making them easier to read while leaving intact the main arguments, doctrines,
and lines of thought.
</pre>

will result in:

        On this site you will find versions of some classics of early modern philosophy, and a few from the 19th

        century, prepared with a view to making them easier to read while leaving intact the main arguments, doctrines,

        and lines of thought.

<code> tag

The <code> tag is a phrase tag. It defines a piece of computer code.

E.g.:
<code> On this site you will find versions of some classics of early modern philosophy, and a few from the 19th
century, prepared with a view to making them easier to read while leaving intact the main arguments, doctrines,
and lines of thought.
</code>

will result in:
On this site you will find versions of some classics of early modern philosophy, and a few from the 19th
century, prepared with a view to making them easier to read while leaving intact the main arguments, doctrines,
and lines of thought.

<tt> tag

The <tt> tag defines teletype text. Not Supported in HTML5.

E.g.:
<tt>Teletype text</tt>

will result in:
Teletype text

<kbd> tag

The <kbd> tag is a phrase tag. It defines keyboard input.

E.g.:
<kbd>Keyboard input</kbd>

will result in:
Keyboard input

<var> tag

The <var> tag is a phrase tag. It defines a variable.

E.g.:
<var>Variable</var>

will result in:
Variable

<sup> tag

The <sup> tag defines superscript text.
Superscript text appears half a character above the normal line, and is sometimes rendered in a smaller font.

E.g.:
This text contains <sup>superscript</sup> text

will result in:
This text contains superscript text

<sub> tag

The <sub> tag defines subscript text.
Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font.

E.g.:
This text contains <sub>subscript</sub> text

will result in:
This text contains subscript text

<blockquote> tag

The <blockquote> tag specifies a section that is quoted from another source.

E.g.:
<blockquote>
On this site you will find versions of some classics of early modern philosophy, and a few from the 19th
century, prepared with a view to making them easier to read while leaving intact the main arguments, doctrines,
and lines of thought.
</blockquote>

will result in:

On this site you will find versions of some classics of early modern philosophy, and a few from the 19th
century, prepared with a view to making them easier to read while leaving intact the main arguments, doctrines,
and lines of thought.

<q> tag

The <q> tag defines a short quotation.

E.g.:
<q>Variable</q>

will result in:
Variable

<ins> tag

The <ins> tag defines a text that has been inserted into a document.

E.g.:
<ins>Variable</ins>

will result in:
Variable

<del> tag

The <del> tag defines text that has been deleted from a document.

E.g.:
<del>Variable</del>

will result in:
Variable

<address> tag

The <address> tag defines the contact information for the author/owner of a document or an article.

E.g.:
<address> Minsk
Coolman street 15-13
</address>

will result in:

Minsk
Coolman street 15-13

<nobr> tag

The <nobr> tag element prevents the text it contains from automatically wrapping across multiple lines,
potentially resulting in the user having to scroll horizontally to see the entire width of the text. Not supported in HTML5

E.g.:
<nobr>
On this site you will find versions of some classics of early modern philosophy, and a few from the 19th
century, prepared with a view to making them easier to read while leaving intact the main arguments, doctrines,
and lines of thought.
</nobr>

will result in:
On this site you will find versions of some classics of early modern philosophy, and a few from the 19th century, prepared with a view to making them easier to read while leaving intact the main arguments, doctrines, and lines of thought.

<wbr> tag

The <wbr> (Word Break Opportunity) tag specifies where in a text it would be ok to add a line-break

E.g.:
To learn AJAX, you must be familiar with the XML fffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffff<wbr>Http<wbr>Request Object.

will result in:
To learn AJAX, you must be familiar with the XML fffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffHttpRequest Object.
